HTML Tag Reference

Tag Name Description
<!--...--> Comment Comment which will be not seen on document.
<a> Hyperlink Link which is clickable to go to different part of page, different page or site.
<button> Button Clickable button as a link.
<img> Image Tag which helps to insert image to document.
<li> List item One item of ordered or unordered list.
<span> Inline section Inline section of document.

CSS Properties Reference

Name Example Description
Display display: none; Element is hidden.
Border border: 5px; Element has 5px border around.
Position position: fixed; Element stays in the same place even if page is scrolled.
Z-index z-index; -1; Element stays under element which has higher number.
Float float: left; Element moves to left side of document.
